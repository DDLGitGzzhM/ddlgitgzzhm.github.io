市场需要什么

网易 :

  1. 熟悉 Gin、Go-zero
  2. 熟悉 Restful API 设计规范
  3. 熟悉 Golang 生态的服务架构设计
  4. 熟悉 中间件的使用 MySQL、Redis、Kafka、RocketMQ
  5. 具有高并发业务相关经验

游卡 :

  1. 精通 Golang 相关网络服务开发 熟悉 Python
  2. 熟练掌握设计模式以及面向对象的设计模式
  3. 熟悉网络基本原理，比如 TCP/IP，HTTP 协议
  4. 熟悉 MYSQL 数据库，了解 MYSQL索引优化，查询优化，存储优化

字节 :

  1. 熟悉分布式系统设计、微服务系统设计、稳定性治理、常用中间件原理

腾讯 :

  1. 熟悉 Mysql、redis、es、mongo等存储 以及常见的消息中间件
  2. 熟悉 TCP/IP 与 HTTP 等网络协议
  3. 有海量服务、高性能、分布式系统设计、开发经验者优先
  4. AI 工程化实践 : 熟悉使用 主流 Ai coding工具，有LLM接口继承，对话系统开发或Ai Agent 工程化落地经验者优先
